FSpawn

by TheRockYT



You need to have Java 8 or newer installed.



What is FSpawn?

FSpawn is a simple system for spigot servers to set spawns and warps.



Features

- First-Join-Teleport

- Multible spawns

- Warps



TeleportMode

- 0: FirstJoin

- 1: Always

- 2: Never



Installation

- Download the latest version

- Move the file into the plugins folder of the spigot server

- Restart the server



Commands

/spawn {name} - Teleports you to a spawn

/setspawn [name/"default"] {permission} - Set a spawn

/deletespawn [name/"default"] - Delete a spawn

/fspawn reload - Reload the plugin



Permissions

FSpawn.SET - Set a spawn

FSpawn.DELETE - Delete a spawn

FSpawn.RELOAD - Reload the plugin
